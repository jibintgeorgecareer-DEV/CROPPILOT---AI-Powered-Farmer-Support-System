CropPilot is a web-based agricultural support platform designed to help farmers make better farming decisions using Artificial Intelligence. The system integrates Machine Learning (ML) and Deep Learning (DL) techniques to provide smart recommendations and expert support for farmers.

Features
🌿 Crop Disease Detection

Farmers can upload images of crop leaves. A Convolutional Neural Network (CNN) model analyzes the image to detect possible crop diseases and provides treatment suggestions.

🌱 Crop & Fertilizer Recommendation

Farmers can enter information such as soil type, water availability, and environmental conditions. Based on these inputs, AI/ML models recommend suitable crops and fertilizers to improve productivity.

📈 Market Price Insights

The platform shows current and predicted market price trends, helping farmers plan when to sell their produce.

💬 Farmer–Officer Communication

Farmers can directly message Agriculture Officers through the platform to receive expert advice and support.

Goal

CropPilot aims to improve agricultural decision-making, reduce crop losses, and enhance productivity by combining AI-powered insights with expert guidance in a simple and accessible digital platform.
